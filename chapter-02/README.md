## 📁 File Commands

```bash
cp file1 file2      # Copy
mv file1 file2      # Move/rename
rm file             # Remove
rm -rf dir          # Force remove recursively
find . -name "*.txt" # Find files
locate filename     # Quick find
```

## 🔗 Links

```bash
ln file hardlink    # Hard link
ln -s file symlink  # Symbolic link
```
