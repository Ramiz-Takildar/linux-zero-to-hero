# Chapter 5 Interview Questions

## Q1: ps aux columns meaning?
**A:** PID, USER, CPU%, MEM%, VSZ, RSS, TTY, STAT, START, TIME, COMMAND

## Q2: Difference between kill and kill -9?
**A:** kill sends SIGTERM (graceful), -9 sends SIGKILL (force)

## Q3: What is systemd?
**A:** Linux system and service manager (replaces init)
